# Neo-Brisnet Admin Dashboard

Administrative interface for managing products, orders, and users.

## Features

- Product management (CRUD)
- Order management and refunds
- User management
- Credit ledger administration
- Analytics and reporting

## Development

```bash
npm run dev
```

Access at: http://localhost:3002

- Bulk product import
- Creator royalty reports
